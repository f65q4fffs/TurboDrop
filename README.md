# TurboDrop
Un mod utilitaire pour Minecraft 1.21.1 (NeoForge) permettant d'éjecter instantanément le contenu d'un coffre ou d'une case unique d'inventaire au sol. Gère la détection automatique des touches (QWERTY/AZERTY), les très gros stacks (>64) sans lag ni perte d'objets, et s'intègre parfaitement avec le mod Configured en jeu.
